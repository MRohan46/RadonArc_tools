Number Systems & Operations

Number Systems

DEC: Decimal (base 10) - 0-9
HEX: Hexadecimal (base 16) - 0-9, A-F
OCT: Octal (base 8) - 0-7
BIN: Binary (base 2) - 0-1

Bitwise Operations

AND: Both bits must be 1
OR: At least one bit is 1
XOR: Bits are different
NOT: Inverts all bits

How to Use

Select a number system (DEC/HEX/OCT/BIN), enter your number, and see instant conversions to all other systems. Use bitwise operations for programming calculations.

Common Questions

How do I convert decimal to hexadecimal?

Enter your decimal number while in DEC mode. The HEX value will automatically display above. Click HEX to switch to hexadecimal mode.

What does AND mean in programming?

AND is a bitwise operation that compares two numbers bit-by-bit. The result bit is 1 only if both input bits are 1. Example: 5 AND 3 = 1 (binary: 101 AND 011 = 001)

Why can't I enter 9 in binary mode?

Binary only uses digits 0 and 1. Numbers 2-9 are disabled in binary mode. Similarly, in octal mode (base 8), only 0-7 are available.

What are hexadecimal letters A-F?

In hex, A=10, B=11, C=12, D=13, E=14, F=15. These extend the base-10 system to base-16.

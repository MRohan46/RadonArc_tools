About Hexadecimal Numbers

What is Hexadecimal?

Hexadecimal (base-16) uses 16 digits: 0-9 and A-F, where A=10, B=11, C=12, D=13, E=14, F=15. It's commonly used in computing as a human-friendly representation of binary data.

Common Uses

• HTML color codes (#FF0000)
• Memory addresses
• MAC addresses
• Assembly language
• Debugging and programming

Conversion Formula

Multiply each digit by 16 raised to its position power (from right, starting at 0). Example: 1A3 = (1×16²) + (10×16¹) + (3×16⁰) = 419

Common Questions

How do I convert hex to decimal manually?

Write the digits, multiply each by 16 raised to its position power (starting from 0 on the right), then sum all results. For example, 2F = (2×16¹) + (15×16⁰) = 32 + 15 = 47.

What does FF in hex represent in decimal?

FF in hex equals 255 in decimal. It's the maximum value for a single byte (8 bits) and is commonly used in HTML color codes for full red, green, or blue.

Why are letters used in hex?

Letters A-F represent values 10-15 because we need single digits for base-16. Without letters, we'd have to write 10 as two digits, which would be confusing.

Is hex case-sensitive?

No, both uppercase and lowercase are accepted. FF and ff both represent 255. This converter accepts both formats.

About Binary Numbers

What is Binary?

Binary (base-2) uses only two digits: 0 and 1. It's the fundamental language of computers, where each digit represents a bit (binary digit).

Common Uses

• Computer processing
• IP addressing
• Digital electronics
• Bitwise operations
• Data storage (bits/bytes)

Conversion Method

Repeatedly divide by 2 and read remainders from bottom to top. Example: 13 ÷ 2 = 6r1 → 6÷2=3r0 → 3÷2=1r1 → 1÷2=0r1 → 1101

Common Questions

How do I convert decimal to binary manually?

Divide the decimal number by 2 repeatedly, writing down the remainders (0 or 1) from each division. Read the remainders from last to first. Example: 13 ÷ 2 = 6r1, 6÷2=3r0, 3÷2=1r1, 1÷2=0r1 → 1101.

What is 255 in binary?

255 in binary is 11111111 (eight 1s). This is the maximum value for a single byte (8 bits) and is common in IP addresses (255.255.255.0).

Why do computers use binary?

Computers use binary because transistors have two states: on (1) and off (0). This makes binary the most reliable and efficient way to represent data electronically.

What's the difference between bit and byte?

A bit is a single binary digit (0 or 1). A byte is 8 bits, which can represent numbers from 0 to 255. Our converter shows the binary representation and its place values.
